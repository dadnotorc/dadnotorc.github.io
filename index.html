掌握你的在线时间：Apple Screen Time 设置指南

发表于 2024-02-08

介绍

电脑和智能手机在给我们带来遍历的同时，也给我们提出了一个问题，如何防止沉迷于数字世界，怎样才能做到适度使用。这个问题对于家长来说，可能更加重要。现在的孩子或多或少都会接触到智能电子产品，作为没有什么自制力的他们，更需要家长来引导。

Screen Time by Apple

Screen Time 是苹果公司推出的一项功能，旨在帮助用户更好地管理和控制他们在 iOS 设备上的使用时间。它提供了一系列工具和功能，让用户了解他们的设备使用情况，并帮助他们设定时间限制和目标，以实现更健康、更平衡的数字生活。

以下是 Screen Time 的主要功能和特点：

  • 应用程序使用报告：Screen Time 会记录并显示用户每个应用程序的使用时间，包括每天和每周的使用情况。这使得用户可以清楚地了解自己在哪些应用程序上花费了大量时间。
  • 应用程序限制：用户可以使用应用程序限制功能，设置对特定应用程序的时间限制。这对于限制孩子在游戏或社交媒体应用上的时间非常有用，以确保他们的使用时间得到合理的控制。
  • 屏幕时间限制：Screen Time 还允许用户设置对整个设备的屏幕时间限制。用户可以设定每天或每周的屏幕时间目标，并在达到限制后自动禁用设备或特定应用程序的使用。
  • 设备使用目标：用户可以设定设备使用目标，比如每天使用设备的最长时间，或者在特定时间段内使用设备的时间。Screen Time 会提醒用户接近或超过目标时的情况。
  • 家庭共享：对于家庭用户，Screen Time 支持家庭共享功能，允许家长对孩子的设备使用进行监控和管理。

通过这些功能，Screen Time可以帮助用户更好地了解和控制他们在iOS设备上的使用时间，从而促进更健康、更平衡的数字生活。

如何设置 Screen Time

激活Screen Time：步骤指南，告诉读者如何在iOS和macOS设备上启用和配置Screen Time功能。
创建限制和目标：演示如何设置应用程序限制、屏幕时间限制和设备使用目标。

在iOS设备上设置：

  1. 打开设置：在你的iOS设备上打开“设置”应用程序。

  2. 找到 Screen Time：滚动并点击进入“屏幕使用时间”选项。

  3. 启用 Screen Time：如果是第一次使用 Screen Time，系统会要求你启用该功能。按照屏幕上的指示进行操作。

  4. 设置：在 Screen Time 设置页面，你可以看到你的设备使用情况摘要。你可以点击查看详细报告。

  • “停用时间”：设置每日机器停用时间段

    • “停用期间组织是用设备”： 该选项将禁止 app 的使用（始终允许的 app 除外）

  • “App 限额”：可以针对某一个，或者某一类 app 设置具体的使用时间限额

    • 记得要打开 “达到限额是阻止使用”

  • “使用允许”：这里可以设置一些在任何时间段都可使用的 app，例如相机，照片，短信，等等。。。

  • 除此以上三项设置之外，还有另外三项设置，可根据需求设置

    • “限定通信”：设置可以与之通信的联系人
    • “通信安全”：检查短信中的敏感照片
    • “内容与隐私限制”：设置 app 年龄限制，隐私，等。。。
  1. 家庭共享：如果你在家庭共享中有孩子的设备，你可以设置监控和管理孩子的设备使用情况。

在 macOS 设备上设置：

  1. 打开系统偏好设置：点击屏幕左上角的苹果图标，选择“系统偏好设置”。

  2. 找到 Screen Time：在系统偏好设置中，找到并点击“屏幕时间”。

  3. 启用 Screen Time：如果是第一次使用 Screen Time，系统会要求你启用该功能。按照屏幕上的指示进行操作。

  4. 设置使用时间：在 Screen Time 设置页面，你可以看到你的设备使用情况摘要。你可以点击查看详细报告。

  5. 设置应用限制、设置屏幕时间限制和 设置设备使用目标：设置步骤与在 iOS 设备上类似，在对应的选项下设置你的偏好。

  6. 家庭共享：同样，如果你在家庭共享中有孩子的设备，你可以在 macOS 中设置监控和管理孩子的设备使用情况。

其他非 Apple 设备

VirtualBox虚拟机无法识别网银U盾的解决方案

发表于 2024-02-08

Situation

主机: macOS Ver. 10.15.6
虚拟机: VirtualBox Ver. 6.1.12 + Windows 10

  • 在主机上插入招商银行23版UKey, 在Win VM中可以看到一个 USB TOKEN设备. 但是挂载时提示错误.
  • 在VBox settings -> Ports -> USB -> USB Device Filters中尝试加载新设备时, 得到的是Vendor ID与Product ID均为0000的unknown device.

解决方案

  1. 确认VBox已安装Oracle VM VirtualBox Extension Pack
  2. 在主机上插入UKey
  3. 访问 About This Mac -> System Report -> Hardware / USB, 找到名为Token的设备
    • 记录Vendor ID与Product ID
  4. 在VBox settings中, 修改USB device filter, 填入
    • Name = Token
    • Vendor ID
    • Product ID
  5. 拔出UKey
  6. 启动Win VM
  7. 插入UKey
