    掌握你的在线时间：Microsoft Family Safety 中屏幕时间管理设置指南

    发表于 2024-02-08

    Microsoft Family Safety

    承接上文，如果你使用的是 Windows PC，微软也有一款家庭安全应用程序，名叫 Microsoft Family Safety，旨在帮助家长和监护人监控和管理他们家庭成员的在线活动，以确保他们的安全和健康。以下是该应用的一些主要特点和功能：

    1. 设备活动监控：
    • 屏幕时间管理：家长可以查看每个家庭成员在设备上花费的时间，并设定每日使用时间限制。
    • 应用程序和游戏限制：家长可以设置哪些应用程序和游戏可以被访问，以及允许的年龄级别。
    1. 网络活动监控：
    • 网页浏览控制：家长可以过滤和阻止不适合孩子的网站，以及设置搜索过滤器来限制搜素结果。
    • 活动报告：家长可以查看家庭成员的在线活动报告，包括浏览历史记录、搜索记录等。
    1. 安全提醒和警告：
    • 活动提醒：家长可以接收有关家庭成员的特定在线活动的提醒和警告，如使用时间过长、访问不安全的网站等。
    • 地理围栏和位置共享：家长可以设置地理围栏，并在家庭成员离开或进入指定区域时接收通知。
    1. 家庭连接和通信：
    • 家庭成员管理：家长可以添加和管理家庭成员的帐户，并分配不同的权限和设置。
    • 通信和共享：家庭成员可以使用该应用程序进行家庭内的通信和共享，包括消息、日历事件、位置等。
    1. 跨平台支持：
    • 多设备支持：Microsoft Family Safety 可在 Windows、iOS 和 Android 设备上使用，家庭成员可以随时随地访问和管理他们的安全设置。

    附加说明一下， Microsoft Family Safety 提供了基本的免费功能，但也提供了额外的高级功能，需要通过 Microsoft 365 订阅来获取。以下是其免费功能和额外的订阅功能：

    免费功能：

    • 屏幕时间管理：可以查看每个家庭成员在设备上花费的时间，并设置每日使用时间限制。
    • 网页浏览控制：可以过滤和阻止不适合孩子的网站，以及设置搜索过滤器来限制搜索结果。
    • 活动提醒：接收有关家庭成员的特定在线活动的提醒和警告，如使用时间过长、访问不安全的网站等。
    • 家庭成员管理：可以添加和管理家庭成员的帐户，并分配不同的权限和设置。
    • 跨平台支持：可以在 Windows、iOS 和 Android 设备上使用，家庭成员可以随时随地访问和管理他们的安全设置。

    额外的订阅功能（需要 Microsoft 365 订阅）：

    • 应用程序和游戏限制：可以设置哪些应用程序和游戏可以被访问，以及允许的年龄级别。
    • 活动报告：可以查看家庭成员的在线活动报告，包括浏览历史记录、搜索记录等。
    • 地理围栏和位置共享：可以设置地理围栏，并在家庭成员离开或进入指定区域时接收通知。
    • 通信和共享：可以使用该应用程序进行家庭内的通信和共享，包括消息、日历事件、位置等。

    如何设置

    创建 Microsoft 家庭组：

    1. 登录 Microsoft 帐户：在 Windows 上登录你的 Microsoft 帐户。
    2. 打开家庭安全页面：在浏览器中打开 Microsoft Family Safety 页面。
    3. 创建家庭组：在页面上找到“家庭成员”或“家庭”选项，并选择“创建家庭组”。
    4. 邀请家庭成员：按照页面上的指示，邀请你的家庭成员加入家庭组。他们会收到邀请邮件，并按照指示加入你的家庭组。
    5. 开始监控：一旦设备加入家庭组，你就可以开始监控和管理他们的活动了。

    在家长或监护人帐户上设置屏幕时间限制

    1. 登录 Microsoft Family Safety：在浏览器中打开 Microsoft Family Safety 页面，使用家长或监护人的 Microsoft 帐户登录。
    2. 选择家庭成员：在页面上选择你要设置屏幕时间限制的家庭成员。
    3. 设置 Windows 屏幕时间限制：在 Windows 屏幕时间设置页面，你可以看到一个时间表，你可以在这里设置屏幕时间限制。
      • 选择每日屏幕时间：在时间表上选择每个时段的屏幕时间限制。你可以设置不同的限制，比如平日和周末的限制可以不同。
      • 设定可用时间：你可以设置每天的可用时间段，家庭成员只能在这些时间段内使用设备。
      • 特定应用程序限制：你还可以针对特定的应用程序设置限制，比如游戏或社交媒体应用。
    4. 保存设置：设置完成后，记得点击保存以确保设置生效。

    在家长或监护人帐户上设置网页浏览限制

    1. 登录 Microsoft Family Safety：在浏览器中打开 Microsoft Family Safety 页面，使用家长或监护人的 Microsoft 帐户登录。
    2. 选择家庭成员：在页面上选择你要设置屏幕时间限制的家庭成员。
    3. 设置网页浏览限制：点击 Edge 标签进入 Microsoft Edge Web 安全性设置页面。（该页面的选项仅适用 Microsoft Edge 浏览器）
      • 启用网页过滤器：在网页浏览设置页面中，启用网页过滤器功能。你可以选择使用 Microsoft 的标准过滤器，也可以自定义过滤器来限制特定类型的网站内容。
      • 设置允许或阻止的网站：根据需要，你可以手动添加允许或阻止的网站列表。点击“允许列表”或“阻止列表”并添加你想要设置的网站。
    4. 保存设置：设置完成后，记得点击保存以确保设置生效。

    其他设置方式

    除了在网页上设置以外，也可以通过其他方式完成设置：

    • 在 PC 设备上打开 Microsoft Store 应用商店，搜索并安装 Microsoft Family Safety 应用
    • 在 iPhone/Android 上下载 Microsoft Family Safety 应用
